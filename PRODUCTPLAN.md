A booking app allowing users to book sessions with an esthetician.

## Features

- A welcome message
- Ability to switch between English and Spanish
- A way to see gallery of work from the esthetician
- Booking sessions with an esthetician using a calendar and form to fill in personal information
- A place to leave anonymous reviews

## Tech stack

### Frontend

- Svelte
- SvelteKit
- Typescript
